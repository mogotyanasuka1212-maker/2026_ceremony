# 2026年 木更津市二十歳を祝う会 特設Webサイト

令和8年（2026年）1月11日に開催される「木更津市二十歳を祝う会」の案内用特設Webサイトです。
参加者（特に木更津第三中学校出身者）に向けた、当日のスケジュール、座席表、写真撮影の流れなどを分かりやすく周知するために作成しました。

## 🌐 公開URL
https://mogotyanasuka1212-maker.github.io/2026_ceremony/
## ✨ 特徴
* **レスポンシブデザイン**: スマートフォンでも見やすいレイアウト。
* **カウントダウンタイマー**: 開催日時までの残り時間をリアルタイム表示。
* **ハンバーガーメニュー**: 各ページへスムーズに移動できるナビゲーション。
* **現在地のハイライト**: メニュー内で「今どのページにいるか」を強調表示。
* **画像拡大機能**: 座席表や案内図をタップして拡大表示可能。
* **UX向上機能**: 「トップへ戻るボタン」や「アコーディオン式FAQ」の実装。
* **検索避け設定**: 関係者のみの共有を想定し、検索エンジンにインデックスされない設定(`noindex`)済み。

## 📂 ファイル構成
* `index.html`: トップページ（開催概要、注意事項、カウントダウン）
* `member.html`: 実行委員メンバー紹介
* `access.html`: 会場へのアクセス、地図、バス時刻表、駐車場案内
* `schedule.html`: 当日の式次第
* `notes.html`: 入退場、座席表、写真撮影の流れ
* `faq.html`: よくある質問、お問い合わせ、同窓会リンク
* `style.css`: サイト全体のデザイン設定
* `script.js`: カウントダウン、メニュー、スクロール動作などの制御
* `images/`: 座席図、駐車場マップなどの画像素材

## 🛠️ 使用技術
* HTML5
* CSS3 (Flexbox, Grid, Animation)
* JavaScript (Vanilla JS)
